HOW TO USE THE DOCKER APP
1. DOWNLOAD AND INSTALL DOCKER
2. OPEN A TERMINAL/CMD AND NAVIGATE TO THE DIRECTORY. E.G. C:\Users\YOURUSERNAME\ads-group-project\docker-app-rent-predictor
3. BUILD THE IMAGE VIA docker-compose up --build
4. ONCE IT IS FINISHED BUILDING OPEN ANOTHER TERMINAL/CMD AND NAVIGATE TO THE SAME FOLDER
5. LAUNCH THE APP VIA docker-compose run predictor

IF YOU WANT TO USE YOUR OWN FILES TO TRAIN THE MODEL MAKE SURE YOU FOLLOW THE SAME FORMAT AS IN THE data_immo_geocoded.csv FILE
